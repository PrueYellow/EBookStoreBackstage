 EBookStoreBackstage


【書本管理功能介紹】
書本資訊匯入功能
會自動新增出版商與作者資料
書本如果有重複會提示出全部書籍已存在訊息

【書本進貨管理功能介紹】
進貨資料匯入功能
進貨資料新增後會取得資料匯入時間
單一入庫與刪除功能
入庫後會更新庫存數量
一鍵入庫與一鍵刪除
